# Introduction
This repo contains a web-based dashboard to the Colonies framework based on React, React-Bootstrap, and Admin-LTE3 for React.

## To start it
```console
npm install
npm start
```
Use the login credential below:

Username: admin@colonyos.io
Password: admin
